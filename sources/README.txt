HƯƠNG DẪN CÀI ĐẶT

Bước 1: Để cài đặt dùng lệnh: npm install

Bước 2: Vào thư mục: src/configs/connectStr.js, chỉnh sửa thông tin để kế nối database

Bước 3: Để chạy chương trình dùng lệnh: npm start

                password: 123
